🧮 Hesap Makinesi (Tkinter)

Python ve Tkinter kullanılarak geliştirilmiş iOS tarzı modern bir hesap makinesi uygulamasıdır.
Basit ve gelişmiş modları, geçmiş kayıt sistemi ve modern arayüz tasarımı içerir.

✨ Özellikler
🧮 Basit hesap makinesi modu
🔬 Gelişmiş (bilimsel) mod (sin gibi fonksiyonlar)
📜 Geçmiş kayıt sistemi (timestamp ile)
🗑️ Seçili geçmişi silme
🧹 Tüm geçmişi temizleme
💾 Geçmişi dosyaya kaydetme (history.txt)
🎨 iOS tarzı modern koyu tema
🔘 Canvas ile yapılmış yuvarlak butonlar
⚡ Akıcı ve modern kullanıcı arayüzü

🛠️ Kullanılan Teknolojiler
Python 3
Tkinter (GUI arayüzü)
math modülü
datetime modülü
🚀 Kurulum ve Çalıştırma
1. Projeyi indir
git clone https://github.com/ismaileyg/hesap-makinesi-pro.git
2. Klasöre gir
cd hesap-makinesi-pro
3. Uygulamayı çalıştır
python hesap_makinesi.py
